How to Generate Your CSV Data

To use this dashboard, you need a Google Search Console CSV export.

  1. Log in to Google Search Console.
  2. Go to Performance > Search results.
  3. Set the date range (e.g., Last 3 Months).
  4. Switch to the Pages tab below the chart.
  5. Click Export > Download CSV.
  6. Extract the ZIP and locate Pages.csv.
  7. Upload that file using the button above.
