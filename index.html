How to Generate Your CSV Data

To use this dashboard, you need to provide a Google Search Console CSV export.

  1. Log in to Google Search Console.
  2. Go to the Performance > Search results report.
  3. Ensure you have the Dates set to a meaningful range (e.g., Last 3 Months).
  4. Switch the view to the Pages tab below the chart.
  5. Click the Export button (top right) and select Download CSV.
  6. Extract the ZIP and locate the Pages.csv file.
  7. Upload that file here using the button in the top right.
